Account Deletion

ACCOUNT AND DATA DELETION REQUEST

App: Trinacria Barberia Siciliana

Users can request account deletion and data removal through the following process:

1. In-App Deletion (Soft Delete)
Users can delete their account directly from the application. This action disables the account but does not permanently remove all data from the system.

2. Full Data Deletion Request
For complete and permanent deletion of all associated data, users must contact the data controller via email:

Email: trinacriabarberiasiciliana@gmail.com

Please include:
- Registered email address
- Request for full account and data deletion

3. Data That May Be Deleted
- Personal profile information (name, surname, email, phone number)
- Authentication data
- Booking history
- Device tokens

4. Data That May Be Retained Temporarily
- Security logs (if required)
- Backup systems (for limited time before full purge)

Retention is strictly limited to operational and security requirements.

5. Processing Time
Deletion requests are typically processed within a reasonable timeframe after verification of ownership.
